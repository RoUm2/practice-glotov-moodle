## Чтобы запустить сервер Moodle сначала потребуется открыть файл docker-compose.yml в редакторе VS-code, после потребуеся открыть терминал git-bash и прописать следующую команду:
    # docker-compose up -d
## После ввода данной команды ждем пока поднимутся контейнеры и переходим в Docker и в развернутом контейнере переходим по локальному порту, оказываясь на сайте Moodle.
