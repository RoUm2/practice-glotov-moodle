Лекция 5. Интернет, открытое ПО и современные тенденции
Тема: Развитие компьютерных сетей, интернета, возникновение открытого программного обеспечения и современные вызовы (облака, ИИ, квантовые вычисления).
Цель урока: Рассмотреть эволюцию от единичных компьютеров к глобальной сети и понять современные направления развития вычислительной техники.
Время: 20 мин.
Ключевые понятия: ARPANET, World Wide Web, GNU/Linux, облачные вычисления, искусственный интеллект, квантовый компьютер.

1. Введение
С 1970‑х годов компьютеры начали объединять в сети. ARPANET, созданный по заказу Министерства обороны США, стал прообразом интернета. В 1990‑х Тим Бернерс‑Ли изобрёл World Wide Web, сделав интернет доступным для всех. Параллельно развивалось движение открытого программного обеспечения (GNU, Linux), которое изменило индустрию. Сегодня мы наблюдаем переход к облачным вычислениям, взрывной рост искусственного интеллекта и эксперименты с квантовыми компьютерами.

2. Основная часть
ARPANET (1969): первая сеть с коммутацией пакетов, использовала протокол NCP, позже заменённый на TCP/IP.

World Wide Web (1989): HTML, HTTP, URL — технологии, сделавшие интернет гипертекстовой средой.

Открытое ПО: Linux (1991) — ядро, распространяемое под лицензией GPL, стало основой для серверов, Android, суперкомпьютеров.

Облачные вычисления: AWS (2006) — модель предоставления вычислительных ресурсов по запросу.

Современные направления: ИИ (нейросети, большие языковые модели), квантовые вычисления (кубиты, сверхпроводящие процессоры).

Пример кода: простейший HTTP‑сервер на Python (демонстрация современного подхода к веб‑технологиям):

python
from http.server import HTTPServer, BaseHTTPRequestHandler

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        self.send_response(200)
        self.end_headers()
        self.wfile.write(b'Hello, history of computing!')

server = HTTPServer(('localhost', 8080), Handler)
server.serve_forever()
