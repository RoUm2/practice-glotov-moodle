Лекция 4. Микропроцессоры и рождение персонального компьютера
Тема: Третье и четвёртое поколения ЭВМ: интегральные схемы, микропроцессоры, появление ПК.
Цель урока: Проследить, как микропроцессор сделал компьютеры доступными для индивидуального использования, и познакомиться с первыми массовыми моделями.
Время: 20 мин.
Ключевые понятия: интегральная схема, микропроцессор, Intel 4004, Altair 8800, Apple II, IBM PC, MS-DOS, ассемблер.

1. Введение
Изобретение интегральной схемы (1959) позволило разместить сотни транзисторов на одном кристалле. В 1971 году Intel выпустила первый микропроцессор 4004 — «компьютер на одном чипе». Это дало старт эпохе персональных компьютеров. В 1975 году Altair 8800 предложил недорогой набор для сборки, а в конце 1970‑х Apple II и IBM PC сделали компьютеры привычными в домах и офисах.

2. Основная часть
Микропроцессор: объединяет арифметико‑логическое устройство и управление на одном кристалле.

Altair 8800: продавался в виде набора для сборки; управлялся переключателями на передней панели. Именно для него Билл Гейтс и Пол Аллен написали интерпретатор Бейсика.

Apple II (1977): готовый компьютер с цветной графикой, клавиатурой и дисководом — первый успешный ПК для массового рынка.

IBM PC (1981): открытая архитектура, использовал процессор Intel 8088 и операционную систему MS-DOS. Задал стандарт де‑факто для ПК.

Программирование на ассемблере: для микропроцессоров популярным был ассемблер (Z80, 8086). Пример вывода символа на экран в MS-DOS (прерывание 21h).

Пример кода на ассемблере для Intel 8086 (вывод символа 'A'):

assembly
MOV AH, 02h   ; функция 02h – вывод символа
MOV DL, 'A'   ; символ для вывода
INT 21h       ; вызов прерывания DOS
MOV AH, 4Ch   ; завершение программы
INT 21h
3. Пример/практика
Простая программа на ассемблере для эмуляции Z80 или 8086 (или в виде псевдокода) — сложение двух чисел из памяти и вывод результата в шестнадцатеричном виде (идея):

text
; Предположим, числа в ячейках 100h и 101h
MOV AL, [100h]   ; загрузить первое число
ADD AL, [101h]   ; прибавить второе
MOV [102h], AL   ; сохранить результат
; Далее код вывода (опущен для краткости)
